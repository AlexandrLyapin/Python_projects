Телеграм бот для покупки энергетиков, написанный на Python при помощи библиотеки telebot.

ТГ бота: @energymiptbot

Для начала работы с ботом нужно отправить команду "/start".
По команде "/help" или "Команды" открывает список команд.

Функционал:
  1) Разделяет запросы пользователей.
  2) Наличие личного кабинета.
  3) Наличие различных команд (список команд можно посмотреть в самом боте).
  4) Наличие изменяющегося каталога товаров (выводит кол-во товара в наличии из текстового файла).
  5) Защита от неправильно совершённых пользователем действий.
  6) Работа с файлами.
